 # Project planner

This project aims to create a simple project planner. The landing page will consist of rounded squares that are in soft colors and has the project titles, the last tile should be one without a title but simply has a plus icon to create a new project.

If you select such a project you will be moved to a similar page which which will be more for "parts" of the project to divide it further into topics. Then in the topics you can create tasks, notes, or other files such as images. Selecting one of these should allow you to view the document or alter it
